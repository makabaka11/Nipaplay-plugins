# 智能弹幕精选（better_danmaku_filter）

NipaPlay 弹幕过滤插件，通过多维度评分智能筛选优质弹幕，过滤低质量内容。

## 功能

- **质量评分**：根据弹幕长度、字符多样性、重复字符比例等维度打分
- **刷屏过滤**：检测短时间内重复发送的相同弹幕
- **相似去重**：基于编辑距离过滤近似重复弹幕
- **短弹幕过滤**：过滤过短的无意义弹幕
- **Emoji 处理**：纯 Emoji 弹幕可选择降权或直接过滤
- **英文/数字惩罚**：全英文/数字弹幕降低评分
- **时间分布优化**：对稀疏时间段的弹幕适当加分，避免某段时间弹幕全部被过滤
- **固定/底部弹幕加分**：优先保留顶部和底部弹幕

## 安装

将 `better_danmaku_filter.js` 导入 NipaPlay 插件系统即可。导入后在插件设置中启用。

## 配置

启用后点击插件旁的扳手图标进入设置，可调整以下参数：

### 数值参数

| 参数 | 说明 | 默认值 | 范围 |
|------|------|--------|------|
| 期望弹幕数 | 目标保留弹幕数（以千为单位），0 表示使用保留比例 | 0 | 0+ |
| 最终保留比例 | 保留弹幕的百分比 | 30% | 5-80 |
| 时间窗口 | 刷屏/去重检测的时间窗口（秒） | 5 | 1-30 |
| 最短弹幕长度 | 弹幕最小字符数 | 3 | 1-10 |
| 重复字符阈值 | 单字符占比超过此值视为重复 | 60% | 30-100 |
| 全英数惩罚 | 全英文/数字弹幕的扣分值 | 30 | 0-80 |

### 开关

| 开关 | 说明 | 默认 |
|------|------|------|
| 过滤相似弹幕 | 编辑距离相似度 > 82% 的弹幕去重 | 开 |
| 过滤刷屏弹幕 | 同一内容在时间窗口内出现 3 次以上标记为刷屏 | 开 |
| 过滤过短弹幕 | 长度小于「最短弹幕长度」的弹幕直接过滤 | 开 |
| 允许纯Emoji降权 | 开启时纯 Emoji 弹幕扣 10 分，关闭时扣 40 分 | 开 |
| 高级语义评分 | 弹幕稀疏时间段加分 + 时间分布优化 | 开 |

## 评分规则

每条弹幕基础分 50 分，根据以下维度调整：

- **长度**：2 字以下 -30，3-5 字 +5，6-15 字 +15，16-30 字 +10，30 字以上 +5
- **字符多样性**：字符种类占比越高加分越多（最高 +20）
- **重复字符**：单字符占比超过阈值时扣分
- **纯 Emoji**：降权 -10 或过滤 -40
- **全英文/数字**：扣分（可配置）
- **弹幕类型**：顶部/底部弹幕 +10
- **稀疏时段**：所在时间段弹幕少于 3 条时 +15

最终按评分从高到低排序，取前 N 条保留。

## 工作流程

```
加载弹幕 → 短弹幕预过滤 → 刷屏检测 → 相似去重 → 质量评分
→ 时间分布优化 → 按比例/期望数截取 → 输出精选结果
```

## 依赖

- NipaPlay >= 1.10.5
- 权限：`danmaku.modify`（修改弹幕显示）、`ui.dialog`（显示提示）
