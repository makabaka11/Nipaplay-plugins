# danmaku_regex_filter — 弹幕正则过滤规则预设

内置常用弹幕正则过滤规则，启用后自动屏蔽匹配的弹幕。

## 包含规则

| 规则 | 说明 |
|------|------|
| 刷屏重复 | 重复字符、笑声刷屏、短语重复、超长弹幕（≥40字） |
| 刷观看次数 | "二刷""三周目"等刷存在感弹幕 |
| 日期考古 | 各类日期格式弹幕（`2026.04.12`、`2026-04-12`、`20260412`、`2026.0412`、`2026年4月12日`）及"考古/留名"等 |
| 加群宣传 | "交流群""加群""QQ群"及带群号数字串的宣传弹幕 |
| 有人在看吗 | "有没有人""有人在看吗"等无意义弹幕 |
| 纯关键词 | "路过""打卡""前排""签到/签"等占位型弹幕及纯短数字 |

## 使用方法

在 NipaPlay 插件设置中启用本插件。每条规则支持独立开关，可在设置页中逐条切换。

## 规则来源

规则参考自社区整理的弹幕过滤方案。
